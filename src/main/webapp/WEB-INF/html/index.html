回收卡常见问题
Common Problem
  • 1、转转卡券支持回收的卡有哪些？
    转转卡券暂时只针对网站公布的卡种进行回收处理，目前支持回收的卡包括电商卡、商超卡、加油卡、游戏卡、话费卡等50余种，如果没有您想要提交的卡，您可以联系客服，后期会逐步地支持更多的卡类，详情请参考网站公示。
  • 3、哪些卡不支持回收？
    凡是已使用的卡、偷盗卡、非正常渠道获得的卡以及网站上没有的卡种都不支持回收。
  • 5、电子卡和实体卡都支持回收吗？
    实体卡是实物的，是一张卡片的形式，卡的背面有卡号和密码；电子卡是虚拟的，是通过网上在线购买获取卡号和密码的；
    实体卡和电子卡进行回收时只需要有卡号和密码即可，不区分电子卡和实体卡。
  • 2、游戏卡与话费卡必看问题！
    游戏卡与话费卡为“自动结算”形式回收，用户提交订单后最快几分钟内即可收到钱款。
    请提交前认真核对选择的面额，如实际面额与选择面额不符，则需要人工干预，不仅有钱款损失的风险，且费时费力！
  • 4、验证成功后，如何收款？
    验证成功并实人认证后，会直接转账到登陆网站的微信账号上。工作时间内提交，一般两个小时内处理完成（特殊情况除外）；非工作时间内，处理时间将延期到下一个工作时间。
  • 6、需要手续费吗？
    在转转卡券进行回收不以任何形式收取手续费，如果有任何收取手续费的行为，请警惕上当受骗。
